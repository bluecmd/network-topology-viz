Network Topology Visualization

A beautiful, interactive 3D network topology visualization component.

Live Demo

How to Use

Add this to your HTML:

<div id="nettopology"></div>
<link rel="stylesheet" href="https://bluecmd.github.io/network-topology-viz/network-topology-viz.css">
<script src="https://bluecmd.github.io/network-topology-viz/nettopology.js"></script>

Optional: Customize with configuration

window.nettopology.init({
    // Optional: Network topology data
    data: {
        nodes: [
            { id: 'node1', position: [1, 0, 0] },
            { id: 'node2', position: [0, 1, 0] }
        ],
        links: [
            { source: 'node1', target: 'node2', trafficIntensity: 'high' }
        ]
    },
    // Optional: Set dark/light mode (default: true for dark mode)
    isDarkMode: true,
    // Optional: Show/hide control panel (default: true)
    showControlPanel: true
});
