Network Topology Visualization

A beautiful, interactive 3D network topology visualization component.

Live Demo

How to Use

Add this to your HTML:

<div id="nettopology"></div>
<script src="https://bluecmd.github.io/network-topology-viz/dist/nettopology.js"></script>

Optional: Customize with data

window.nettopology.init({
    nodes: [
        { id: 'node1', position: [1, 1, 1] },
        { id: 'node2', position: [-1, -1, -1] }
    ],
    links: [
        { source: 'node1', target: 'node2', trafficIntensity: 'high' }
    ]
});
